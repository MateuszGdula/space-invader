# Space Invader

### Control for desktop:
- Move - arrow keys
- Shot - space
- Next weapon - c
- Show menu - esc
- Choose menu item - enter

### Control for mobiles:
- Move - swipes
- Shot - tap game area
- Next weapon - tap on status bar (on the bottom)
- Show menu - back
- Choose menu item - tap

### Suported browsers:
- Chrome
- Firefox
- Chrom mobile
